# SocialMedia-Backend-Node-js-
create Node.js Express social media Rest API using MongoDB. Build a real-world social media app REST API with login, register, and all CRUD operations and how to use MongoDB models with Node.js routers.

# Prerequisites
To run this application, you need to have the following installed on your machine:

Node.js (v14 or higher)
MongoDB
